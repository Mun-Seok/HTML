이미지 태그 : img

img태그는 이미지를 연결시켜 문서에 보이도록 해주는 태그.

img태그는 닫는 태그 없음

링크를 연결할때는 href속성을 쓰고, img태그에 사용되는 속성은 src.

alt라는 속성으로 사용된 이미지에 대한 설명을 입력할 수 있음

위에서 사용된 alt속성은 이미지가 깨지거나, 엑박이 뜨더라도 텍스트로 표현됨

이미지가 잘나온다면 보이지x

img태그는 인라인요소.

작성중인 html 파일과 동일한 위치에 이미지를 불러올경우 : ./파일이름.확장자

작성중인 html 파일과 동일한 위치에 폴더가 있고, 그 안에 있는 이미지를 불러올경우 : ./파일이름.확장자

작성중인 html 파일과 동일한 위치에 폴더가 있고, 그 안에 있는 폴더 안에 있는 이미지를 불러올경우 : ./폴더이름/폴더이름/파일이름.확장자

작성중인 html 파일과 더 상위에 폴더가 있고, 그 안에 있는 이미지를 불러올경우 : ../파일이름.확장자

너비값 : width , 높이값 : height

너비값 높이값 둘중에 하나만 조정을 하면 나머지 크기는 정비율에 맞춰서 자동으로 조정된다.
