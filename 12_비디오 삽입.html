video / source

  1. 비디오 파일을 다운로드 및 저장한 후 사용
  2. 유튜브 등 비디오가 있는 사이트의 링크를 복사해서 사용
  3. 저장된 영상 파일을 삽입하는 경우 한글이 없어야 한다.
  4. 파일 용량이 커질수록 화질은 좋아진다. 대신에 페이지 로딩 속도가 느려진다.

  1. video 태그

    • source 태그를 video 태그 안에 넣고, src 속성으로 삽입할 동영상 파일의 경로를 입력하여 사용
    • controls : 컨트롤 패널을 표시해주는 속성(재생, 일시정지, 볼륨 등등)
    • width / height :영상의 가로 / 세로 사이즈를 정해주는 속성
    • autoplay : 자동재생 시켜주는 속성
    • muted : 음소거 시키는 속성
    • loop : 반복재생을 시켜주는 속성
    • 켜자말자 자동재생되는데 소리 없이 영상만 계속 나오는 경우 : autoplay와 muted를 동시에 사용!

iframe

  1. 영상의 HTML 문서 주소를 넣어서 사용
  2. < iframe src="영상주소" title="설명" width="너비" height="높이">< /iframe>
  3. 유튜브 같은 경우 동영상에 오른쪽 클릭하면 "소스복사"를 클릭해서 옮기면 정말 친절하게 iframe까지 다 복사해서 옮겨진다.
  4. 간혹 가다가 이런식으로 끌어다 쓰지 못하게 막혀있는 영상들이 있다. 그거는 끌어와봐야 확인이 가능하다
